## IT Asset Management System - Asset Registry & Tracking

| ID    | Module                    | User Story                                                                                                                                                                                                                              | Acceptance Criteria (AC)                                                                                                                                                                                                                                                              |
| ----- | ------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| AR-1  | Asset Registry & Tracking | As an IT Asset Manager, I want to register a new asset by first selecting its Type (Hardware or Software), so that the form only prompts me for relevant fields and ensures data accuracy.                                              | - The form uses conditional logic: selecting 'Hardware' displays Hardware fields; selecting 'Software' displays License/Software fields.<br>- I can register the asset with a unique Asset ID                                                                                         |
| AR-2  | Asset Registry & Tracking | As an IT Asset Manager, I want to store key Hardware details including Serial Number, Model, RAM/Storage, Vendor, Purchase Date, and Initial Cost, so that I have a complete physical and financial record of the equipment.            | - The form includes mandatory fields for Serial Number, Model, Purchase Date, and Cost.<br>- The form includes fields for technical specs (RAM, Storage, Processor Type).                                                                                                             |
| AR-3  | Asset Registry & Tracking | As an IT Asset Manager, I want to store key Software details including License Type, Number of Seats, License Key, and Renewal Cost, so that I can maintain compliance and manage subscription payments.                                | - The form includes mandatory fields for License Type (e.g., Perpetual, Subscription) and Number of Seats.<br>- Fields for License Expiration Date and Renewal Cost are available.                                                                                                    |
| AR-4  | Asset Registry & Tracking | As an IT Technician, I want to import assets in bulk using a CSV or Excel file, so that I can onboard large quantities of standardized equipment efficiently.                                                                           | - The system provides a downloadable import template.<br>- The system processes the file, mapping columns to asset fields, and generates an error report for failed records.                                                                                                          |
| AR-6  | Asset Registry & Tracking | As a Finance Analyst, I want to tag each asset with its correct Cost Center and Business Unit during registration, so that I can allocate costs and calculate depreciation accurately.                                                  | - Mandatory fields for Cost Center and Business Unit are linked to pre-defined lookup lists.                                                                                                                                                                                          |
| AR-7  | Asset Registry & Tracking | As an IT Asset Manager, I want to assign an asset to a specific Employee and Location, so that I can establish clear custody and maintain accountability for company property.                                                          | - The system allows linking an asset to an Employee ID and a Physical Location (e.g., "4th Floor Office").<br>- The asset status automatically updates from 'Available' to 'Assigned'.                                                                                                |
| AR-8  | Asset Registry & Tracking | As an Employee, I want to view a list of all assets currently assigned to me upon login, so that I can easily verify my current responsibilities for company equipment.                                                                 | - A personal dashboard or profile view displays all assets linked to the logged-in user.                                                                                                                                                                                              |
| AR-9  | Asset Registry & Tracking | As an IT Technician, I want to use a Check-Out/Check-In feature for assets, so that I can accurately track the movement of equipment and update its current status and custodian automatically.                                         | - The system logs the date, time, new custodian (or Project Name), and the Technician performing the transaction.                                                                                                                                                                     |
| AR-10 | Asset Registry & Tracking | As an Admin, I want to define and save new Asset Categories (e.g., "Developer Laptop," "Network Gear") and their corresponding required fields, so that the registration process is standardized across different types of equipment.   | - The system provides an interface for Admins to create new templates and assign which fields are mandatory/optional for that category.                                                                                                                                               |
| AR-11 | Asset Registry & Tracking | As an Admin, I want to search, filter, and sort the asset inventory by Status, Department, Type, Model, and Cost Center, so that I can quickly locate assets for audit or reallocation purposes.                                        | - The inventory list supports complex filtering by combining multiple criteria (e.g., 'Assigned' Hardware in 'Marketing').                                                                                                                                                            |
| AR-12 | Asset Registry & Tracking | As a Compliance Officer, I want to export filtered asset data (e.g., all high-value hardware assigned to a specific department) to a spreadsheet, so that I can easily share up-to-date inventory lists for auditing.                   | - The system provides an export function that includes all asset details from the current filtered view in CSV/Excel format.                                                                                                                                                          |
| AR-13 | Asset Registry & Tracking | As an IT Asset Manager, I want to filter assets based on their Check-Out/Check-In history, so that I can identify equipment that is frequently moved or those that have been assigned for too long.                                     | - Filtering criteria includes "Days Since Last Check-In" or "Total Times Checked Out".                                                                                                                                                                                                |
| AR-14 | Asset Registry & Tracking | As a Security Officer, I want to search for assets based on their Serial Number or Asset ID using a barcode scanner interface, so that I can quickly verify an item's details at a physical security checkpoint.                        | - The system recognizes input from a standard barcode/RFID scanner and immediately navigates to the corresponding asset page.                                                                                                                                                         |
| AR-15 | Asset Registry & Tracking | As an Admin, I want to define and save frequently used filter configurations (e.g., "Available Laptops" or "Licenses Expiring Soon"), so that I can access critical reports quickly without manually recreating the filters every time. | - The system allows saving a filter combination with a custom name for one-click reuse.                                                                                                                                                                                               |
| AR-16 | Asset Registry & Tracking | As a system user, I want the system to validate asset serial numbers and IDs so that duplicate assets cannot be accidentally registered.                                                                                                | - System enforces unique values for "Serial Number" and "Asset ID" fields.<br>- Attempting to save a record with an existing Serial Number triggers an error message.<br>- Validation occurs in real-time or upon form submission.                                                    |
| AR-17 | Asset Registry & Tracking | As an asset manager, I want to update asset details whenever changes occur (status, location, ownership) so that the system always reflects the latest information.                                                                     | - "Edit" button is available on the Asset Profile page for authorized users.<br>- System logs the "Last Updated By" and timestamp automatically.<br>- Critical fields (e.g., Asset ID) are read-only; operational fields are editable.                                                |
| AR-18 | Asset Registry & Tracking | As a system user, I want to reassign an asset to a different employee or department so that organizational changes are reflected accurately.                                                                                            | - A "Reassign" action is available on the asset record.<br>- Selecting a new Employee/Department updates the "Assigned To" field.<br>- The previous owner is moved to the "Custody History" log.                                                                                      |
| AR-19 | Asset Registry & Tracking | As an Asset Manager, I want to record vendor details and warranty information so that I can manage support contracts and plan for renewals.                                                                                             | - The asset form includes fields for "Vendor," "Support Phone," and "Warranty Expiry Date".<br>- A dashboard widget alerts the admin of warranties expiring in the next 30 days.<br>- Vendor is a searchable and filterable field.                                                    |
| AR-20 | Asset Registry & Tracking | As an Asset Manager, I want to view the complete custody history of an asset so that I can see all past and present users.                                                                                                              | - A "Custody History" tab is available within the Asset Profile view.<br>- The history log displays a timeline of check-out/check-in events, dates, and employees.<br>- The log is non-editable and serves as an audit trail.                                                         |
| AR-21 | Asset Registry & Tracking | As an Asset Manager, I want to use a barcode scanner to quickly check assets during an audit so that I can verify their existence and condition efficiently.                                                                            | - A dedicated "Audit Mode" provides a full-screen scanning interface.<br>- Scanning an asset's barcode immediately pulls up its profile and allows for a quick status/condition update.<br>- The system generates an audit report listing scanned assets and any found discrepancies. |
| AR-22 | Asset Registry & Tracking | As a Finance Manager I want the system to calculate asset depreciation automatically so that I can maintain accurate financial records.                                                                                                 | - System includes fields for "Purchase Price," "Salvage Value," and "Useful Life."<br>- System calculates "Current Book Value" daily/monthly using Straight-Line depreciation.<br>- Depreciation schedule is viewable in the "Financials" tab.                                        |
| AR-23 | Asset Registry & Tracking | As an IT Administrator I want to mark assets as leased or rented so that I can track return dates and lease costs separately from owned assets.                                                                                         | - "Acquisition Type" dropdown includes options for: Owned, Leased, and Rented.<br>- Selecting "Leased" makes "Lease Expiration Date" and "Return Date" mandatory fields.<br>- The system triggers an alert 30 days before the "Return Date."                                          |
| AR-24 | Asset Registry & Tracking | As an Asset Manager, I want to search for assets using criteria like Asset ID, Serial Number, and Name, so that I can quickly locate specific asset records.                                                                            | - The global search bar is accessible from the main navigation header.<br>- Search accepts partial matches (e.g., last 4 digits of a serial number).<br>- Results display a list view with columns: Asset Name, ID, Serial Number, and Status.                                        |
| AR-25 | Asset Registry & Tracking | As an Auditor, I want to filter and sort the asset list by Asset Status (e.g., In Use, Retired), Department, and Asset Type, so that I can easily review subsets of assets for inventory and compliance checks.                         | - Filters for Status, Department, and Asset Type are available as multi-select checkboxes.<br>- Users can sort the results list by any column header.<br>- A "Clear All Filters" button resets the view.                                                                              |
